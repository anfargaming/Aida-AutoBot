# Install

## Step 1

```bash
 git clone https://github.com/anfargaming/Aida-AutoBot.git
```
```bash
 cd Aida-AutoBot
```

## Step 2

```bash
npm install axios
```

## Step 3

```bash
nano token.txt
```
F12 - Aplication - Local Storage - Access Token

## Step 4

```bash
node aida.js
```
